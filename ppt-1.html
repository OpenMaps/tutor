Formats for spatial data

  1. File-based data
    • Shapefile, KML, GeoJSON, & TopoJSON, gpx, csv
  2. Spatially-enabled database data
    • Proprietary: Microsoft SQL Server, Oracle Spatial, and the Esri ArcSDE middleware.
    • Open equivalents: PostgreSQL with PostGIS extension, SQLite with SpatiaLite extension.
  3. Raster formats
    • Open raster formats JPEG, PNG, TIFF, BMP, GIF.

2. Script.js setup and Map initialization

  • Initialize the map and set the center point
  • Add a base layer to the map
  • Import data and add it to the map
  • Add event handlers

var map = new L.Map('map', { 
center: new L.LatLng(-2.652, 36.8325),
zoom: 13,
minZoom: 12,
maxZoom: 16
});

2. Script.js setup: Background layer

remove the 'center' and 'zoom' options from the L.map constructor

center and zoom level with the setView() method


var osm = L.tileLayer('http://{s}.tiles.wmflabs.org/bw-mapnik/{z}/{x}/{y}.png', {
maxZoom: 18,
attribution: '© OpenStreetMap'});

map.addLayer(osm);

//or

L.tileLayer('http://{s}.tile.osm.org/{z}/{x}/{y}.png', {
attribution: '© OpenStreetMap contributors'
}).addTo(map);

2. Script.js setup: Background layer

little more user friendly


var map = L.map('map',{scrollWheelZoom:false}).setView([-2.652, 36.8325], 13);
L.tileLayer('http://{s}.tile.osm.org/{z}/{x}/{y}.png', {
attribution: '© OpenStreetMap contributors'
}).addTo(map);

2. Script.js: Add a WMS Layer to the Map

using L.tileLayer.wms() displays a WMS service as a tile layer on the map


var tileUrl = 'https://{s}.tile.thunderforest.com/cycle/{z}/{x}/{y}.png?apikey=',
layer = new L.TileLayer(tileUrl, {maxZoom: 18});

3. Script.js: Add a Default Marker to the Map

creates a marker variable and assigns it a marker object using L.marker()


var marker = L.marker([-2.652, 36.8325]).addTo(map);

3. Script.js: Add a Custom Marker to the Map

creates the ‘MyIcon’ variable and assigns it an icon marker using L.icon()

A few options are passed to L.icon():

  • iconUrl: the url to an image to be used as icon marker
  • iconSize: thesize to make the icon marker

var MyIcon = L.icon({
iconUrl: 'http://localhost/svg/flame.svg',
iconSize: [38, 95], // size of the icon
popupAnchor: [0,-15]
});
var marker = L.marker([-2.652, 36.8325], {icon: MyIcon}).addTo(map);

3. Event handlers

  • Set a trigger and action
  • Triggers: click, mouseover, map load, popup open...
  • Actions: an existing method or your own function
  • Example: Show name when user clicks on the icon

//in js
L.marker([-2.652, 36.8325], {icon: MyIcon}).bindPopup('Nairobi').addTo(map);
//in css
.custom .leaflet-popup-tip,
.custom .leaflet-popup-content-wrapper {
background: #e93434;
color: #ffffff;
}

Basic Elements of Javascript

  • Each line should ends with ; (semicolon).
  • // -­‐‐ comment -­‐‐ everything after is ignored.
  • var variable -­‐‐ creates a variable that can hold a value (label,identifier).
  • Equal sign after a variable = -­‐‐ assign a value on the right to a variable.
  • Single values -­‐‐ numbers 50.51, string -­‐‐ ‘text’
  • Grouped values: arrays [1,2,3], objects/dictionaries {x:1,y:2}

Leaflet’s API

  • L (capital l) variable is how you access Leaflet’s functionality.
  • L. (capital l dot) -­‐‐ means that you’re using Leaflet’s API.
  • For example: to “test” a method with two values you would write something like: L.method(value1,value2).
  • Method is a function related to an object (capital L) that returns a value..
  • It often requires a value or values to work .
  • Every task/action in Leaflet has a method.
